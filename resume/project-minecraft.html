Project Description

Led a team project to build and operate a Minecraft server where users can join and play minigames. The project focused on deploying a working environment, integrating plugins, improving gameplay through iterative prototypes, and collecting user feedback through surveys.

  • Set up an Ubuntu environment and installed a Minecraft server management panel (Pterodactyl).
  • Configured connectivity so players could access the server and the team could manage it remotely.
  • Integrated plugins and supported updates across multiple prototype versions.
  • Helped troubleshoot issues and supported deployment of custom gameplay changes.
  • Created and conducted surveys to measure user experience and guide improvements.

Skills Learned

  • Linux administration fundamentals (setup, services, basic troubleshooting).
  • Network connectivity concepts for hosted services and basic access control thinking.
  • Team-based deployment workflow and change tracking across versions.
  • User research methods: survey design, control group feedback, iteration based on results.
  • Project leadership: sprint planning, task coordination, and deliverable ownership.
