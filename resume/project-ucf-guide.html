Project Description

Served as project manager for a class project creating a student-friendly guide to help new UCF students find and use campus resources. The final flyer/pamphlet was improved through multiple versions and testing, then shared with students and submitted to student government.

  • Managed tasks and workload using Trello to coordinate team progress and deadlines.
  • Researched campus opportunities and services and helped write readable, engaging content for new students.

  • Supported distribution and collected positive student feedback.
